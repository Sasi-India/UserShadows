UserShadows by whytheplatypus

Experiments with processing and Kinect user tracking

How it works

I had the pleasure of finding a great library to work with Processing and the Kinect. Even better it works on the mac! I've enjoyed working with processing before, it's a simple front end for java so the learning curve involved is minimal and it allows designers and non designers alike come up with good looking and quick little applications.

The projects

At the moment this processing sketch contains two similar projects using the Kinect and it's ability to keep track of the space occupied by a user. They were inspired by the RAM exhibit at THEMUSEUM in Kitchner, ON. In particular the Trash Mirror by Daniel Rozin which created shadows using servo mounted trash.

Both projects in the sketch choose one user and minipulate their pixels in some way. The first project centers the user on the screen so that no matter where they are in the room they are the center of attention for the project. This is an interesting disorienting effect since it's natural to expect the shadows to be relatively accurate and having a single outlire (someone who's in a different place on the screen than in real life) is off putting.

The second project chooses one user and flips their pixels around their center of mass. This has some interesting effects again because the representation is almost like the real world with only one user changed. It's also fun because if the user walks normally they will appear to moonwalk on the screen!
